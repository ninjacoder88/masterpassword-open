### FINDING-001 · HIGH — Missing Authorization on Password Generation Endpoint

| Field | Value |
|---|---|
| File | `/src/MasterPassword/Controllers/API/v1/PasswordController.cs` line 15 |
| OWASP | A01 - Broken Access Control |
| CWE | CWE-285 |

**Evidence:**

```
[HttpGet]
```

**Description:** The Get endpoint in PasswordController is missing authorization, allowing any unauthenticated user to generate passwords.

**Attack Scenario:** An attacker can make unlimited requests to the password generation endpoint, potentially exhausting server resources or generating passwords for malicious purposes. This could lead to denial-of-service or abuse of the password generation service.

**Remediation:** Add the [Authorize] attribute to the Get method in PasswordController.cs to require authentication before allowing password generation.
